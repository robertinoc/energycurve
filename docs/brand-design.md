# Brand & Design

EnergyCurve keeps a dark, premium product language built around performance
intelligence rather than generic DJ software tropes.

## Core Direction

- dark or dark-leaning surfaces
- strong typography hierarchy
- restrained neon accents
- editorial spacing with product-led structure
- subtle glow and hover states instead of loud effects

## Core Palette

- Background: `#0B0B0F`
- Surface: `#1A1A22`
- Violet: `#7B3FE4`
- Cyan: `#00D1FF`
- Accent pink: `#FF2D75`
- Monochrome fallback: `#F3F7FF`

## Typography

- Headings: `Space Grotesk`
- Body: `Inter`
- Technical values: `Geist Mono`
